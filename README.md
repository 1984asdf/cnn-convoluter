# cnn-convoluter

An interactive player for CNN convolution

https://pwwang.github.io/cnn-convoluter/

![cnn-convoluter](./cnn-convoluter.gif)

## Building your own version
```shell
# clone this repo
# install dependencies
npm install
# run a dev-server
npm run dev
# or build the app
npm run build
```
